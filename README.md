# 使用方法

# 注意

接口与[ms-ra-forwarder](https://github.com/wxxxcxx/ms-ra-forwarder) 相同，即在阅读APP中直接将url改为 `http://ip:1233/api/azure或者ra`

微软Azure接口: `ip:1233/api/azure`
Edge大声朗读接口: `ip:1233/api/ra`
